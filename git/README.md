### リモートリポジトリ clone してローカルリポジトリを作成する
```
git clone https://github.com/ユーザ名/リポジトリ名.git
```

### リモートリポジトリの変更内容を取り込む
```
git pull
```
pull = fetch + merge

## branch
branchの確認や作成を行うコマンド
### 現在のブランチを確認する
```
git branch
```

### ブランチを作成する
```
git branch [ブランチ名]
```

## switch
### ブランチを切り替える
```
git switch [ブランチ名]
```
